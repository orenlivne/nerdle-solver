# Nerdle Solver
Nerdle Solver (for 5..8 slots).
Mini Nerdle = 6 slots
Nerdle = 8 slots

## Prerequisites
* C++ compiler (`gcc`; if different, modify the `CC` variable in Makefile` accordingly.)

## Installation
* Clone the git repo.
* conda env create -f environment.yml
* python setup.py build_ext --inplace && make build
